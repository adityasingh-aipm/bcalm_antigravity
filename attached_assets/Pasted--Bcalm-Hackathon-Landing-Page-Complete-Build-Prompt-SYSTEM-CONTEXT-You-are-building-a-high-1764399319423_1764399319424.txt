## TECHNICAL REQUIREMENTS

### Frontend Stack
- React 18 with TypeScript
- Vite (build tool)
- Tailwind CSS (styling)
- Shadcn UI (component library)
- React Hook Form (form management)
- Axios (API calls)

### Features to Implement
1. **OTP Verification**:
   - Phone input with validation
   - Modal popup for OTP entry
   - 6-digit code input with auto-focus
   - Resend button (30-second cooldown)
   - Error handling for invalid OTPs

2. **Form Validation**:
   - Name: Min 3 characters, letters only
   - Phone: 10 digits, Indian numbers (+91 optional)
   - Gmail: Valid email format, must end with @gmail.com
   - Company/College: Required, autocomplete suggestions

3. **Success State**:
   - Show confirmation message
   - Display countdown to next hackathon
   - Option to share on LinkedIn
   - Form clears after submission

4. **UTM Tracking**:
   - Capture ?utm_source=linkedin&utm_campaign=hackathon params
   - Store with registration data

5. **Responsive Design**:
   - Mobile-first approach
   - Fully functional on iPhone SE to iPad
   - Touch-friendly buttons (min 44px height)
   - Optimized font sizes for mobile

### Backend Integration Points
- **POST /api/register**: Register user (name, phone, email, company)
- **POST /api/send-otp**: Trigger OTP to phone number
- **POST /api/verify-otp**: Verify OTP and confirm registration
- **GET /api/hackathon-status**: Get next hackathon date/time
- **POST /api/send-confirmation-email**: Send thank-you email
